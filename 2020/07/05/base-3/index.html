左侧固定，右侧自适应

  • float + margin

或者利用BFC
把main元素变成BFC，left就不会占用main的空间，左右隔开

  • position + margin
  • flex

flex 布局的item其中
flex： a b c

a: flex-grow: 存在剩余空间是否放大
b: flex-shrink: 空间不够是否要缩小
c: flex-basics: 不缩小放大的占据主轴空间大小
