执行环境

执行环境定义了变量或函数有权访问的其他数据，决定了它们各自的行为。
分成：

  • 全局执行环境

  • 函数执行环境

  • 每个执行环境都有三个组件：*

  1. 执行上下文，this就指向执行上下文对象
  2. 词法环境
  3. 变量环境

执行上下文

分成全局执行上下文和函数执行上下文
js执行的时候，有一个执行上下文栈，在遇到新的函数执行就会把新的执行函数的上下文推到执行上下文栈里，执行结束弹出

词法环境对象

环境记录器
环境记录器分为两种：

  1. 声明式环境记录器，存在于函数作用域中，存储变量、函数、参数。
  2. 对象式环境记录器，存在于全局作用域和块级作用域中，存储变量、函数。
