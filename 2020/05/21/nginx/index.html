nginx 编译

安装的nginx目录下执行 make && make install + 编译文件需要放的路径

nginx 启动 暂停 重载

  • 启动
    nginx的编译后文件内执行：
    ./sbin/nginx -c conf/nginx.conf

  • 操作
    nginx -s
    nginx -s stop 立刻停止服务
    nginx -s quit 优雅停止服务(进行中的web请求执行完在结束)
    nginx -s reload 重载配置文件
    nginx -s reopen 重新开始记录日志文件

    停止一般使用stop

    改了nginx.conf内容之后，执行nginx -s reload 就行了
